## 🚀 Featured Projects

### [Cliq](https://github.com/rivando-al-rasyid/cliq-frontend) / [Backend](https://github.com/rivando-al-rasyid/cliq-backend)

Full-stack short link management platform.

**Tech:** React, Vite, React Router Data Mode, Tailwind CSS, DaisyUI, Go, Gin, PostgreSQL, Redis, JWT, HttpOnly Cookies, Docker

**Highlights:**

* Built a short link platform with user registration, login, protected dashboard, profile page, and link management
* Implemented React Router loaders and actions for route protection and form submission
* Used HttpOnly cookie authentication instead of storing JWT tokens in localStorage
* Added custom slug validation, reserved slug protection, link deletion, and short URL copy flow
* Built backend REST API for authentication, profile management, password reset, short link creation, dashboard pagination, soft delete, and public slug redirect
* Added PostgreSQL migrations, Redis connection support, Swagger documentation, and Docker Compose setup

### [VanWallet](https://github.com/rivando-al-rasyid/vanwallet-react) / [Backend](https://github.com/rivando-al-rasyid/vanwallet-backend)

Full-stack wallet management dashboard and REST API.

**Tech:** React, Vite, Redux Toolkit, React Router, Tailwind CSS, DaisyUI, Chart.js, Go, Gin, PostgreSQL, Redis, JWT, Docker, Nginx

**Highlights:**

* Built wallet dashboard pages for balance summary, income, expense, and recent transactions
* Implemented authentication pages including login, register, forgot password, reset password, and logout
* Created transaction flows for top-up, transfer, withdrawal, expense, and transaction history
* Added chart filters for 7-day, 14-day, and 30-day financial views
* Built backend API for authentication, profile management, wallet transactions, receiver search, dashboard summary, and reports
* Improved security flow by using HttpOnly cookie authentication, token digest storage, PIN lockout handling, and pending top-up webhook-ready flow
* Added Docker, Docker Compose, PostgreSQL migrations, Redis, Swagger documentation, and deployment health checks

> Note: VanWallet is a portfolio and learning project. It is not intended for real financial transactions without proper security audit, compliance review, monitoring, and legal approval.
